# Plugin Claude - Arthur Loyd Bretagne

Plugin Claude Desktop (Cowork) avec les skills immobilier commercial pour Arthur Loyd Bretagne.

## Skills inclus

| Skill | Description |
|-------|-------------|
| `al-offres` | Creation et gestion des offres immobilieres |
| `al-charte-graphique` | Charte graphique Arthur Loyd (couleurs, typographies, logos) |
| `al-prospection` | Prospection et commercialisation immobiliere |
| `al-etats-des-lieux` | Workflow etats des lieux entree/sortie |
| `al-newsletters` | Creation de newsletters MJML immobilieres |

## Installation

Dans Claude Desktop (Cowork) :

1. Aller dans **Plugins** > **Parcourir les plugins** > **Personnel**
2. Cliquer sur **+** pour ajouter une marketplace
3. Entrer `Rod292/PluginClaudeAL`
4. Installer le plugin **arthur-loyd-bretagne**
